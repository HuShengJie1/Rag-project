# 表格统一风格决策说明

本文件说明本次全文表格统一所采用的最终版式标准。统一标准以全文多数表格已有的主流写法为基础，并优先参考第 4 章表格的呈现风格，不额外发明新的视觉体系。

## 一、识别出的主流风格

统一前，全文多数表格已经具备以下共同特征：

1. 表格均使用 `booktabs` 风格的横线。
2. 表格均居中排版。
3. `\caption` 均位于 `\label` 之前。
4. 表格均不使用竖线。
5. 多数表格为控制宽度统一采用了整体缩放策略。

因此，本次统一不改变这些主流特征，而是在此基础上补齐字号、行距、注释和缩放逻辑。

## 二、最终统一标准

### 1. 表内字号

- 统一采用：`\small`
- 应用方式：
  - 每张表在 `tabular` 外层使用局部组 `{ ... }` 包裹；
  - 在该局部组内统一声明 `\small`。

选择理由：

- 统一前大多数表格没有显式字号，导致字号由缩放比例间接决定；
- `\small` 既能保证表格可读性，也能兼容多数宽表的版面需求；
- 相比 `\footnotesize`，` \small ` 更接近正文常见学位论文表格风格，不会显得过小。

### 2. 表格行距

- 统一采用：`\renewcommand{\arraystretch}{1.2}`

选择理由：

- 统一前仅有个别表格显式设置了行距，其余表格默认较紧；
- 1.2 倍行距能在不显著拉高表格高度的前提下改善可读性，适合作为全文统一标准。

### 3. caption 风格

- 统一采用：
  - 保持 `\caption` 在前、`\label` 在后；
  - 通过 `\captionsetup[table]{font=small, skip=6pt}` 统一表题字号与表题下间距。

选择理由：

- 全文原本 caption/label 顺序已一致；
- 因此本次只做微调，不改动表题表达内容与命名方式。

### 4. 边框风格

- 统一采用：`booktabs`
- 具体要求：
  - 使用 `\toprule`、`\midrule`、`\bottomrule`；
  - 不引入竖线；
  - 不改动现有表格数据和语义结构。

选择理由：

- 全文所有表格本就采用 `booktabs`，一致性最好；
- 与学术论文常见规范一致，且观感整洁。

### 5. 注释风格

- 统一采用宏：
  - `\thesistablenote{...}`
- 实际格式：
  - `\vspace{0.5ex}\par\raggedright\small\textit{注：...}`

选择理由：

- 全文只有两张表带注释，但写法原本为手工内联，后续容易漂移；
- 抽成统一宏后，注释字号、斜体、对齐方式和上下间距均可保持一致。

### 6. 过宽表格处理策略

- 统一采用：
  - `\begin{adjustbox}{max width=\textwidth} ... \end{adjustbox}`
- 不再使用：
  - `\resizebox{\textwidth}{!}{...}`

选择理由：

- 统一前所有表格都使用固定 `\resizebox`，会把窄表强制拉伸至整页宽度，导致不同表格实际字号不一致；
- `adjustbox` 的 `max width=\textwidth` 只在超宽时缩放，不会放大本来就能正常排下的表格，更有利于统一视觉字号。

## 三、未采用的方案

### 1. 未统一改为 `\footnotesize`

原因：

- 虽然更容易让宽表塞进页面，但整体会偏小，不符合当前论文多数表格与正文关系。

### 2. 未强行改写为 `tabularx` 或重新分栏

原因：

- 本次任务重点是全文排版统一，不是重构所有表结构；
- 过度改写列格式会提高误改数据和换行逻辑的风险。

## 四、最终执行策略

本次统一最终执行为：

1. 所有表格统一加入 `\small`；
2. 所有表格统一加入 `\arraystretch=1.2`；
3. 所有表格统一改为 `adjustbox` 控制最大宽度；
4. 所有带注释表格统一改为 `\thesistablenote`；
5. 保持 `booktabs`、caption/label 顺序和表格内容本身不变。
