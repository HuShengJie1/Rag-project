# 本次引用统一规则

## 总体原则

本次处理遵循老师提出的要求：从第三章开始，通常只标注此前从未引用过的参考文献；已经在前文引用过的参考文献，后文原则上不再重复标注。

## 具体执行规则

1. 第一章和第二章中的引用全部保留，作为全文首次引文基础。
2. 从第三章开始，若某条参考文献 key 已在第一章或第二章出现，则原则上删除其后续重复引用。
3. 从第三章开始，若某条参考文献 key 第一次出现，则保留。
4. 若一个引用命令中同时包含“旧引用 + 新引用”，仅删除旧引用，保留新引用。
5. 若一个引用命令中的所有 key 都属于应删除的旧引用，则删除整个引用命令，并同步检查句内空格和标点是否正常。
6. 若删除旧引用后会导致句子缺少必要支撑、语义容易歧义，允许保留重复引用，并在 `kept_repeated_citations.md` 中说明原因。

## 本文实际应用结果

经全文审计，真正落入“第三章及之后重复引用前两章旧文献”的情况只有 1 个正文句子，其中包含 2 个相同的重复引用命令：

- 文献 key：`gao_retrieval-augmented_2024`
- 位置：第 3 章第 333 行

该处删除后，句意、标点与 LaTeX 语法均保持正常，因此按规则直接删除。
