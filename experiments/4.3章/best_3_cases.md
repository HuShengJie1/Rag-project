# 4.3 章节最终入选的 3 个案例

## 案例 C1

- 案例编号：C1
- 查询内容：试卷命题与前两届试题的重复率限制是多少？
- 单阶段 Top-1 / Top-3 结果摘要：Top-1 为 `附件1-9#3`，命中“ A、B 两套试卷中重复题目不得超过 30\% ”；Top-3 内虽已包含 `附件1-9#2` 的正确条款，但首位证据对应的是另一种“重复率”约束对象。
- 双阶段 Top-1 / Top-3 结果摘要：Top-1 调整为 `附件1-9#2`，直接命中“试题与前两届试题不得重复超过 30\%”；相邻但不充分的 `#3` 被保留在次位。
- 文档名与页码：`附件1-9  信息学院试卷命题及课程目标达成度评价实施细则`，第 1 页（OCR 回查）。
- 局限性类型：语义相关但证据不足
- 简短分析：单阶段检索能够感知“重复率 30\%”这一语义特征，但无法精确区分“前两届试题”与“A、B 两套试卷”两类相邻规则；重排序后，真正对应问题约束对象的条款被提升至首位。

## 案例 C2

- 案例编号：C2
- 查询内容：开卷考试中直接从教材找答案的题目分值限制是多少？
- 单阶段 Top-1 / Top-3 结果摘要：Top-1 为 `附件1-7#11`，内容是教材编写绩效“最高不超过 600 分”；Top-3 几乎都被“分值/计分”类规则占据，正确证据 `附件1-9#2` 仅排第 4。
- 双阶段 Top-1 / Top-3 结果摘要：Top-1 调整为 `附件1-9#2`，直接给出“直接可以在教材中找到答案的题目分值不应超过总分值的 20\%”；Top-2 为同一细则中的相邻命题条款。
- 文档名与页码：单阶段 Top-1 为 `附件1-7 上海海洋大学信息学院课程教学工作基本规范及考核计算办法` 第 6 页；双阶段 Top-1 为 `附件1-9  信息学院试卷命题及课程目标达成度评价实施细则` 第 1 页。
- 局限性类型：高召回下噪声增多
- 简短分析：在工程认证制度文本中，大量“教材”“分值”“限制”类表述会产生高表层相似度。单阶段检索虽有较高召回，但噪声块反而挤占排序前列；加入重排序后，真正的命题规范条款被优先保留。

## 案例 C3

- 案例编号：C3
- 查询内容：毕业要求和课程体系的合理性评价周期是几年？
- 单阶段 Top-1 / Top-3 结果摘要：Top-1 为 `附件1-14#3`，给出“学生学满四年毕业时完成毕业要求达成度评价”；Top-2 为 `附件1-5#6`，给出“全校性培养方案修订四年一次、课程体系每年可局部调整”；真正与问题主干对应的 `附件1-15#7` 仅排第 3，而课程体系“每两年局部调整”的 `附件1-15#14` 在单阶段仅排第 6。
- 双阶段 Top-1 / Top-3 结果摘要：Top-1 调整为 `附件1-15#7`，明确指出“毕业要求至少每二年综合评价一次”；同时 `附件1-15#14` 升至第 4，使课程体系“每两年局部调整”的证据也进入前列。
- 文档名与页码：关键支撑条款来自 `附件1-15  上海海洋大学信息学院专业教学质量持续改进工作指导意见` 第 2 页与第 3 页；单阶段误导性 Top-1 来自 `附件1-14 上海海洋大学信息学院本科专业毕业要求达成情况评价办法` 第 1 页。
- 局限性类型：跨块/跨页证据割裂
- 简短分析：该问题要求同时满足“毕业要求合理性评价”与“课程体系调整周期”两部分证据。单阶段检索虽然能召回相关条款，但前列结果被相近的“四年周期”条款占据，导致首位证据不具备直接作答性；双阶段排序后，核心证据链更集中、更可用。
